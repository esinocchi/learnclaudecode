## Design Context

### Users
Penn State freshman CS students. They arrive at this site on a laptop, probably at night, trying to figure out how to start a personal project. They have some coding knowledge but feel stuck outside of class. They are curious and motivated but not yet confident. The site needs to feel approachable and credible — like a knowledgeable upperclassman wrote it, not a corporation.

### Brand Personality
Warm, educational, sharp.

- **Warm**: inviting, not clinical. Cream surfaces, not pure white. Typography that reads like a well-made university press book, not a SaaS landing page.
- **Educational**: clear hierarchy, structured layout. The reader should always know where they are and what comes next. Guidance over decoration.
- **Sharp**: confident, not soft. Decisive type choices. No rounded-corner excess. The design makes a point of view clear.

### The One Memorable Thing
This site was built entirely using Claude Code. That fact is the meta-narrative — a guide about using Claude Code to build projects, itself built with Claude Code. This should be stated plainly and prominently, not hidden in a footer. It demonstrates the point rather than just arguing for it.

### Aesthetic Direction
Physical object analogy: a well-typeset university course reader from a professor who actually cares — warm paper, confident headers, clean margins, purposeful structure. Not a textbook. Not a startup landing page.

- **Theme**: Light mode. Warm off-white backgrounds (not pure white).
- **Color**: Deep navy or slate for text. Warm amber/gold as the accent — connects to Claude's warm palette. Subtle warm gray for secondary text.
- **References**: Evan's portfolio at https://www.esinocchi.me/ captures the right personal, thoughtful tone.
- **Anti-references**: Generic edu-tech sites (Canvas, Coursera). Cold blue-heavy SaaS pages. Anything that looks like a WordPress template.

### Design Principles
1. **The meta-narrative is visible** — "Built with Claude Code" is a design element, not a footnote.
2. **Hierarchy earns attention** — spacing and type size do the heavy lifting; no decorative elements that don't also communicate.
3. **Warmth comes from texture, not softness** — warm tones and confident type, not rounded corners and pastel colors.
4. **Students first** — every design decision should reduce friction for someone who has never done this before.
5. **Personal, not institutional** — this is Evan's guide, not a Penn State department page. It has a voice.
